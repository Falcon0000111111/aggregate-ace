# **App Name**: AggregateAce

## Core Features:

- Input Fields: Three input fields for Matric (1100, 10%), Inter (1100, 40%), and MDCAT (200, 50%).
- Percentage Calculation: Calculate the contribution percentage of each field based on the input number and its fixed percentage.
- Aggregate Calculation: Calculate the total aggregate score based on the percentage contributions of each field.
- Result Display: Display the calculated aggregate score in a clear and concise format.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for intellect and precision. It evokes a sense of knowledge and focus.
- Background color: Light grey (#ECEFF1), a desaturated version of the primary, to provide a clean and neutral backdrop.
- Accent color: Indigo (#3F51B5), an analogous color to the primary, is slightly less bright for highlighting key information and buttons.
- Body and headline font: 'Inter', a sans-serif font that provides a modern and neutral aesthetic.
- Clean and simple layout with clear sections for input fields and results.
- Minimalist icons to represent each field (Matric, Inter, MDCAT).
